# Stage 1 — Intent
 
**Goal:** capture what I'm building and what done looks like, in my terms.
 
**Artifact — `docs/INTENT.md`:**
- What I'm building
- Why I want to build it (intrinsic reason is valid; don't justify with markets)
- A concrete, checkable definition of "done"
- Explicit non-goals
**Rules**
- 1.1 Use my words. Don't invent the intent.
- 1.2 Make "done" concrete enough to test against later.
- 1.3 List what this is explicitly NOT trying to do.
- 1.4 Surface every assumption; mark guesses `ASK ME`.
**Tracking:** open a GitHub issue for this stage before starting (G7).
 
**Gate:** write `docs/INTENT.md`, give a ≤5-line summary, STOP. On my approval, read `stage-2-approach.md`.
